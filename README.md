## Tillgängliga script

```
npm install # Installera paket
npm start # Kör lokal utvecklingsmiljö
```

Gå till [http://localhost:3000](http://localhost:3000) i webbläsaren

## Övriga script

```
npm run build      # Kompilera minifierad version
npm run lint       # eslint
npm run lint:fix   # eslint med fix
```
